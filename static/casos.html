Flujo de trabajo

¿Cómo lo integran las agencias?

1

Diseñas el template

Subes la imagen de fondo, posicionas los textos con variables {{first name}} y {campo} y ajustas el estilo visual.

2

Copias el JSON

El editor genera automáticamente el JSON de configuración. Lo copias en un solo clic y lo pegas en tu webhook de ManyChat.

3

ManyChat envía variables

Cada vez que un suscriptor activa el flujo, ManyChat envía su nombre y campos personalizados al API de TextOnFlow.

4

Imagen personalizada en segundos

TextOnFlow genera la imagen con los datos del suscriptor y devuelve la URL lista para enviar por WhatsApp o Messenger.
